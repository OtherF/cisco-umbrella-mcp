# Security Policy

## Supported Versions

The latest release on the `main` branch is actively maintained.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Use [GitHub private vulnerability reporting](https://github.com/OtherF/cisco-umbrella-mcp/security/advisories/new) to report a vulnerability confidentially. You can expect an acknowledgement within 48 hours and a resolution or mitigation plan within 14 days.
